# 音视频处理工具

一个轻便的音频视频快速编辑工具，支持视频转音频、视频剪辑和音频剪辑功能，更便捷的构建音视频数据集。

## 功能特点

### 1. 视频处理
- **视频转音频**: 将视频文件转换为WAV格式的音频文件
- **视频剪辑**: 
  - 支持点击视频画面播放/暂停
  - 可设置剪切起点和终点
  - 支持快进快退（3秒、5秒、20秒）
  - 支持倍速播放（0.5x-2.0x）
  - 可保存多个剪切片段

### 2. 音频处理
- **音频剪辑**:
  - 可视化进度条
  - 支持快进快退
  - 倍速播放功能
  - 精确剪切功能
  - 批量保存剪切片段
